HIL-Debugger是一个基于多智能体协作与硬件在环（HIL）架构的嵌入式设备自动化联调系统。它通过**固件代理**、**硬件代理**和**分析代理**三者的协同工作，将传统嵌入式开发中"修改代码 → 手动烧录 → 接线测试 → 抓取日志 → 人工分析"的重复性调试流程，转化为全自动化的闭环流水线。

系统能够自动完成固件编译烧录、硬件激励注入、串口/GPIO/协议数据采集，以及基于长链推理的异常检测与代码级定位，将单轮联调周期从传统的 **45 分钟压缩至 6 分钟**，异常定位效率提升 **85%**。

graph LR
    A[修改代码] --> B[固件Agent<br/>编译 & 烧录]
    B --> C[硬件Agent<br/>激励注入 & 数据采集]
    C --> D[分析Agent<br/>异常检测 & 根因定位]
    D -->|发现问题| A
    D -->|通过| E[输出报告]
